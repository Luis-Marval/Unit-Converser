
# Conversor de Unidades
Programa para convertir Unidades

## Como usar
para poder utilizar esta programa utilizada el siguiente comando:

```build.sh```
Archivo para construir los archivos de produccion

```deploy.sh```
Archivo para iniciar el servidor de programa

## Tecnologias Utilizadas
- NodeJs
- Express
- React
